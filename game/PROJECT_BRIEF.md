# Netbound Project Brief

- Netbound is an offline-first 3D mobile soccer physics puzzle game for iOS and Android.
- Players swipe to shoot a soccer ball through obstacle-course-style levels.
- The most important thing is satisfying, predictable shooting.
- Mobile-first controls, but mouse input must work for desktop testing.
- Keep architecture simple and maintainable.
- Core gameplay must work offline.
- Do not add accounts, servers, multiplayer, ads, purchases, menus, coins, cosmetics, level select, procedural generation, or external assets yet.
- Use typed GDScript where practical.
- Expose gameplay tuning values in the Godot Inspector.
- Make targeted changes only.
- Verify the project after implementing.
