# Netbound Project Brief

- Netbound is an offline-first 3D mobile soccer physics puzzle game for iOS and Android.
- Players swipe to shoot a soccer ball through obstacle-course-style levels.
- The most important thing is satisfying, predictable shooting.
- Mobile-first controls, but mouse input must work for desktop testing.
- Keep architecture simple and maintainable.
- Core gameplay must work offline.
- Build toward a polished offline vertical slice with menus, level select, progression, star ratings, settings, and earnable cosmetic customization.
- Do not add accounts, servers, multiplayer, ads, purchases, analytics, online services, procedural generation, copyrighted assets, or app-store SDKs yet.
- Monetization may be added later, but the offline game must remain independent from it.
- Use typed GDScript where practical.
- Expose gameplay tuning values in the Godot Inspector.
- Make targeted changes in sequential verified phases.
- Verify the real production scene and production input path after implementing.
